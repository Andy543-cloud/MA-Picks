"# MA Picks - Car Rental Application

A full-stack Django web application for car rental booking and management. Users can browse available cars, make reservations, and manage their bookings with a seamless user experience.

## Features

- **User Authentication**: Secure customer signup and login system
- **Car Browsing**: Browse available cars with detailed information and pricing
- **Search Functionality**: Search cars by city location
- **Booking System**: Reserve cars with date selection and payment integration
- **Customer Dashboard**: View booking history and manage reservations
- **Responsive Design**: Mobile-friendly Bootstrap UI with modern styling
- **Profile Management**: Customer profiles with contact information

## Tech Stack

- **Backend**: Django (Python)
- **Frontend**: HTML, CSS, Bootstrap 5
- **Database**: SQLite
- **Features**: Authentication, Search, Payment Integration, Forms Validation

## Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/Andy543-cloud/MA-Picks.git
cd MA-Picks
```

2. Create and activate virtual environment:
```bash
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

4. Run migrations:
```bash
python manage.py migrate
```

5. Start the development server:
```bash
python manage.py runserver
```

Visit `http://127.0.0.1:8000/` in your browser.

## Project Structure

- `home/` - Main Django app with models, views, and templates
- `CarRental/` - Django project settings and configuration
- `home/templates/` - HTML templates for all pages
- `home/static/` - CSS styling and static files
- `home/migrations/` - Database migrations
